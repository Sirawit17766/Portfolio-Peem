# วิธีเปิด Diagram ใน draw.io

ไฟล์ diagram อยู่ที่:

```text
docs/service-architecture.drawio
```

วิธีเปิด:

1. เข้าเว็บ https://app.diagrams.net/
2. เลือก `Device`
3. กด `File`
4. เลือก `Open From`
5. เลือก `Device`
6. เลือกไฟล์ `docs/service-architecture.drawio`

หลังเปิดแล้วสามารถลากกล่อง แก้ข้อความ เปลี่ยนสี หรือ export เป็น PNG/PDF สำหรับใส่สไลด์ได้เลย
